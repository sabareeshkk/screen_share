<h1> Screen Share</h1>
<ul>
	<li>
		$ git clone https://github.com/sabareeshkk/screen_share.git
	</li>
	<li>
		$ cd screen_share
	</li>
	<li>
		$ npm install
	</li>
	<li>
		$ bower install
	</li>
	<li>
		$ npm start
	</li>
</ul>

<p>
	Just start the serve at http://localhost:8000
</p>
<p>
	If you want to test it with https then you can run the server at https://localhost:5000
</p>
